Over

DroidFish is een uitgebreid schaakprogramma, dat gebruik maakt van de zeer sterke Stockfish schaak engine.

Hints

  • Druk lang op de mode/links/rechts knoppen voor extra commando's.
  • Bij het naspelen van een partij kunnen in de spelmodus Bewerken/Naspelen halfzetten teruggenomen worden.

Openingsboeken

DroidFish heeft een klein intern openingsboek. Het programma kan ook externe openingsboeken van het formaat Polyglot of CTG gebruiken:

  1. Kopieër één of meer polyglot boek bestanden naar de DroidFish/book folder op de SD kaart. Polyglot bestanden moeten de extensie .bin hebben. Let op: Android is hoofdletter gevoelig, gebruik .bin en niet .Bin of .BIN.
  2. Kopieër één of meer CTG boek bestanden naar de DroidFish/book folder. Een CTG boek bestaat uit drie bestanden met de extensies .ctg, .ctb en .cto. Alledrie de bestanden moeten gekopieerd worden.
  3. Kies vanuit het programma-menu de optie Selecteer openingsboek.
  4. Selecteer het gewenste openingsboek.

PGN bestanden

DroidFish kan partijen laden die opgeslagen zijn in PGN (Portable Game Notation).

  1. Kopieer de PGN bestanden naar de DroidFish/pgn directory op de SD kaart.
  2. Raak het bord lang aan en selecteer Laad partij vanuit PGN bestand.
  3. Selecteer het bestand en vervolgens de gewenste partij.

Partij-informatie, zoals spelersnamen, kan bewerkt worden door de zettenlijst lang aan te raken en Bewerk partij-informatie te selecteren. Commentaar bij een zet kan bewerkt worden door de zettenlijst lang aan te raken Bewerk Commentaar te selecteren.

PGN bewaren: raak het bord lang aan en selecteer Opslaan als PGN. Selecteer het bestand en vervolgens de positie in de lijst waar de partij moet worden opgeslagen.

N.B. Als de OI File Manager geïnstalleerd is, kan een iedere directory gebruikt worden op partijen op te slaan.

Analyse toevoegen: raak de analyse informatie lang aan en selecteer Analyse Toevoegen. De huidige hoofdvariant wordt toegevoegd als variant in de partijboom.

In analyse modus is het mogelijk om lege zetten toe te voegen aan de partijboom. Let op, lege zetten zijn geen onderdeel van de PGN standaard en zijn daarom misschien niet meer beschikbaar in andere schaakprogramma's die uw PGN bestand openen.

Scid bestanden

Als u Scid on the go heeft geïnstalleerd, is het mogelijk de Scid database files direct vanuit DroidFish te benaderen. Raak het schaakbord lang aan en selecteer Laad partij vanuit Scid bestand.

FEN/EPD bestanden

DroidFish kan posities laden uit FEN/EPD bestanden.

  1. Kopieër de FEN/EPD bestanden naar de DroidFish/epd directory op de SD kaart.
  2. Raak het schaakbord lang aan en kies Positie inlezen.
  3. Selecteer een bestand en vervolgens de gewenste positie.

Het is ook mogelijk om positie in te laden vanuit externe programma's zoals bijvoorbeeld ChessOcr (betaald). Met dit programma kunnen plaatjes van schaakdiagrammen automatisch als positie ingelezen worden.

  1. Raak het schaakbord lang aan en selecteer Positie ophalen.
  2. Volg de verdere instructies van het externe programma.

UCI Engines

DroidFish kan externe UCI engines gebruiken.

  1. Kopieer de UCI engine binary naar de DroidFish/uci directory op de SD kaart. De binaries moeten gecompileerd zijn voor Android.
  2. Kies vanuit het programma-menu de optie Engines beheren en Selecteer Engine om de gewenste engine te activeren.

UCI engines gebruiken vanaf een remote server

DroidFish kan ook UCI engines gebruiken die op een remote server staan.

  1. Installeer chess network server software op de server.
    1. Voor Windows: installeer de server software op de Computer Chess Utility pagina.
    2. Voor Linux, kan mini-inetd uit de tcputils package gebruikt worden.
  2. Selecteer Beheer Engines vanuit het programma-menu en maak een nieuwe netwerk engine aan. Voer de hostnaam en het poortnummer in van de externe engine.
  3. Kies vanuit het programma-menu de optie Engines beheren en Selecteer Engine om de gewenste engine te activeren.

Gaviota eindspel databases

DroidFish kan Gaviota eindspel databases gebruiken:

  1. Kopieer de .gtb.cp4 bestanden naar de DroidFish/gtb directory op de SD kaart.
  2. Kies Endspel Databases instellingen om in te stellen hoe de database gebruikt moet worden.

Syzygy endgame tablebases

To use Syzygy endgame tablebases:

  1. Copy .rtbw and optionally .rtbz files to the DroidFish/rtb directory on the SD card.
  2. Kies Endspel Databases instellingen om in te stellen hoe de database gebruikt moet worden.
