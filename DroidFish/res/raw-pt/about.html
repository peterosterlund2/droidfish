Sobre

DroidFish é uma interface gráfica para xadrez rica em recursos, combinada com o potente software de Xadrez StockFish.

Recursos

  • Livro de aberturas
  • Jogar/Voltar ilimitado
  • Copiar/colar de/para área de transferência
  • Modo de análise
  • Modo para dois jogadores
  • Modo software vs software
  • Definição de posições
  • Amplo suporte a PGN
  • Força de jogo ajustável
  • Softwares UCI de terceiros
  • Tabelas de finais Gaviota
  • Tabelas de finais Syzygy

Dicas

  • Pressione longamente os botões de modo/esquerda/direita para comandos adicionais.
  • Quando estiver jogando novamente uma prtida, altere o modo de jogo para Editar/rejogar partida para poder desfazer/refazer lances.

Livros de aberturas

DroidFish vem com um pequeno livro de aberturas. Se você quiser usar um livro maior, pode configurar o DroidFish para usar livros externos Polyglot ou em formato CTG:

  1. Copie um ou mais arquivos CTG para o diretório DroidFish/book. Livros Polyglot devem ter a extensão de arquivo .bin. Atenção! O sistema de arquivos do Android diferencia maiúsculas de minúsculas, portanto, a extensão deve ser .bin, e não .Bin ou .BIN.
  2. Copie um ou mais arquivos CTG para o diretório DroidFish. Um livro CTG é composto por três arquivos com extensões .ctg, .ctb e .cto. Você deve copiar todos os 3 arquivos.
  3. No menu do programa, ative a função Selecionar livro de aberturas.
  4. Selecione o arquivo de aberturas que deseja usar.

Arquivos PGN

DroidFish pode carregar partidas de arquivos PGN.

  1. Copie um ou mais arquivos PGN para o diretório DroidFish/pgn no cartão SD.
  2. Pressione longamente o tabuleiro e selecione Carregar partida de arquivo PGN.
  3. Selecione o arquivo e depois a partida dentro do arquivo que deseja carregar.

Para editar os cabeçalhos, como nomes de jogadores, pressione longamente a lista de lances e selecione Editar cabeçalhos. Para editar comentários de um lance, pressione longamente a lista de lances e selecione Editar comentários.

Para salvar a edição de sua partida, pressione longamente o tabuleiro e selecione Salvar partida em arquivo PGN. Selecione o arquivo para salvar e em seguida selecione a posição no arquivo onde será salva a nova partida.

Nota! Se o OI File Manager está instalado, qualquer diretório pode ser usado para carregar/salvar partidas.

Para adicionar a análise do software para a árvore da partida, pressione longamente as informações de análise quando o software estiver analisando e selecione Adicionar análise. A variante principal atual é adicionada como uma variante na árvore da partida.

No modo de análise é possível adicionar lances nulos para a árvore da partida. Esteja ciente no entanto de que lances nulos não são suportados pelo padrão PGN e podem não funcionar em outros programas de leitura de arquivos PGN.

Arquivos Scid

Se você tem o Scid on the go instalado, você pode ler arquivos de banco de dados do Scid diretamente a partir do DroidFish. Pressione longamente o tabuleiro e selecione Carregar partidas de arquivos Scid.

Arquivos FEN/EPD

DroidFish pode carregar posições de arquivos FEN/EPD.

  1. Copie um ou mais arquivos FEN/EPD para o diretório DroidFish/epd no cartão SD.
  2. Pressione longamente o tabuleiro e selecione Carregar posição de arquivo.
  3. Selecione o arquivo, e então a posição dentro do arquivo que você deseja carregar.

Você também pode carregar posições de programas externos, tais como o ChessOcr (pago), que permite que você tire uma foto de uma diagrama de xadrez e automaticamente use a posição no DroidFish.

  1. Pressione longamente o tabuleiro e selecione Recuperar posição.
  2. Siga as instruções do programa externo.

Softwares UCI

DroidFish pode usar softwares UCI de terceiros.

  1. Copie um ou mais softwares UCI para o diretório DroidFish/uci no cartão SD. Atenção! Os binários devem ser compilados para o Android.
  2. No menu do programa, escolha Softwares de Xadrez e Selecionar software e escolha o software desejado.

Softwares UCI em servidores remotos

DroidFish pode usar softwares UCI rodando em um servidor remoto.

  1. Instale o servidor de rede de Xadrez no computador remoto.
    1. Para Windows, instale o software de servidor a partir da página Utilitário de Xadrez do computador.
    2. Para Linux, pode ser usado o mini-inetd do pacote tcputils.
  2. Selecione Softwares de Xadrez no menu do programa, crie uma nova configuração de rede e digite o nome do servidor e o número da porta do servidor remoto.
  3. No menu do programa, escolha Softwares de Xadrez e Selecionar software e escolha o software desejado.

Tabelas de finais Gaviota

Para usar tabelas de finais Gaviota:

  1. Copie arquivos .gtb.cp4 para o diretório DroidFish/gtb no cartão SD.
  2. Use a tela de preferências de Tabelas de Finais para controlar como as tabelas são usadas.

Tabelas de finais Syzygy

Para usar as tabelas de finais Syzygy:

  1. Copie os arquivos com extensão .rtbw e opcionalmente .rtbz para o diretório DroidFish/rtb no cartão SD.
  2. Use a tela de preferências de Tabelas de Finais para controlar como as tabelas serão usadas.
