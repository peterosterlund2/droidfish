소개

DroidFish는 기능이 풍부한 체스 그래픽 사용자 인터페이스이며, 아주 강력한 Stockfish 체스 엔진과 결합되어 있습니다.

기능

  • 오프닝 북(Opening book)
  • 제한없는 무르기(undo)/다시 하기(redo)
  • 클립보드로 복사/붙여넣기
  • 분석 모드
  • 2인 플레이어 모드
  • 컴퓨터 vs 컴퓨터 모드
  • 포지션 설정
  • PGN 확장 지원
  • 플레이 강도 조정 가능
  • 서드 파티 UCI 엔진
  • Gaviota 엔드게임 테이블베이스
  • Syzygy 엔드게임 테이블베이스

힌트

  • 추가적인 명령을 위해 '길게 누르기 모드'/ 왼쪽 및 오른쪽 버튼이 사용됩니다.
  • 게임을 다시 플레이할때, 게임 모드를 '싱글 하프 무브'가 가능하도록 '편집/다시 플레이 게임' 모드로 변경하세요.

오프닝 북(Opening books)

DroidFish 는 작은 내장 오프닝 북을 포함하고 있습니다. 더 큰 북(Book)의 사용을 원하시면, DroidFish가 외장 polyglot 또는 CTG book 파일을 사용하도록 설정할 수 있습니다 :

  1. 1개 이상의 polyglot book 파일을 SD 카드상의 DroidFish/book 디렉토리에 복사하세요. Polyglot book은 .bin 파일 확장자를 가져야 합니다. 알림! 안드로이드 파일 시스템은 대소문자 구분에 민감합니다. 따라서 확장자는 반드시 .bin으로 만들어야 하고, .Bin이나 .BIN 은 불가합니다.
  2. 1개 이상의 CTG book 파일을 DroidFish/book 디렉토리에 복사하세요. CTG book은 각각 .ctg, .ctb 그리고 .cto 의 파일 확장자를 가진 세 개의 파일로 구성되어 있습니다. 반드시 세 개 파일 모두를 복사하셔야 합니다.
  3. 프로그램 메뉴에서, 오프닝 북 선택 기능을 활성화하세요.
  4. 사용을 원하는 오프닝 북 파일을 선택하세요.

PGN 파일

DroidFish 는 PGN 파일로부터 게임을 불러올 수 있습니다.

  1. 1개 이상의 PGN 파일을 SD 카드의 DroidFish/pgn 디렉토리에 복사하세요.
  2. 체스판을 길게 누르고 PGN 파일로부터 게임 불러오기 를 선택합니다.
  3. 파일을 선택하면, 게임이 로드됩니다.

플레이어 이름과 같은 게임 머리말(Header)을 수정하기 위해서는, 이동 목록을 길게 누른 후, 머리말 수정 을 선택하세요. 이동을 위한 코멘트(Comment)를 수정하는 경우, 이동 목록을 길게 누른 후, 코멘트 수정 을 선택하세요.

수정된 게임을 저장하려면, 체스 보드를 길게 누른 후, PGN 파일로 게임 저장을 선택하세요. 저장할 파일을 선택 후, 새 게임이 어디에 저장될 것인지 위치를 선택하세요.

알림! 만약, OI File Manager가 설치된 경우, 디렉토리에 구애받지 않고 게임을 불러오거나 저장할 수 있습니다.

게임 트리(tree)에 컴퓨터 분석을 추가하려면, 컴퓨터가 생각하고 있을 때, 분석 정보를 길게 누른 후, 분석 추가 를 선택하세요. 현재의 주요 변수(Principal Variation)가 게임 트리에 변수로 추가됩니다.

분석모드에서는, 빈 이동(Null Move)을 게임 트리에 추가할 수 있습니다. 하지만 빈 이동은 PGN 표준(Standard)에 포함되지 않으며, 다른 PGN을 읽는 체스 프로그램에서는 작동하지 않을 수 있음을 유념하십시오.

Scid 파일

만약, Scid on the go 가 설치되어 있다면, DroidFish로 부터 바로 Scid 데이터베이스 파일을 읽을 수 있습니다. 체스 보드를 길게 누른 후 파일로부터 게임 불러오기 를 선택하세요

FEN/EPD 파일

DroidFish는 FEN/EPD 파일로부터 포지션(Positions)을 불러올 수 있습니다.

  1. 1개 이상의 FEN/EPD 파일을 SD 카드 상의 DroidFish/epd 디렉토리에 복사하세요.
  2. 체스 보드를 길게 누른 후 파일에서 포지션 불러오기를 선택하세요.
  3. 파일을 선택하면, 포지션이 로드됩니다.

ChessOcr(유료) 같은 외장 프로그램으로부터 포지션을 불러들일 수 있으며, 이는 카메라를 사용하여 체스 다이어그램을 찍거나 자동으로 DroidFish의 포지션을 설정할 수 있습니다.

  1. 체스 보드를 길게 누른 후 포지션 찾기 를 선택하세요.
  2. 외부 프로그램의 지시에 따르시기 바랍니다.

UCI 엔진

DroidFish 는 서드 파티 UCI 엔진을 사용할 수 있습니다.

  1. 1개 이상의 UCI 엔진 바이너리를 SD 카드 상의 DroidFish/uci 디렉토리에 복사하세요.
  2. 프로그램 메뉴에서, 체스 엔진 관리 를 선택하고, 체스 엔진 선택 을 누르고, 사용하고자 하는 엔진을 선택하세요.

원격 서버상 UCI 엔진

DroidFish는 원격 서버에서 실행되는 UCI 엔진을 사용할 수 있습니다.

  1. 원격 컴퓨터에 체스 네트워크 서버 소프트웨어를 설치합니다.
    1. Windows는, Computer Chess Utility 페이지에서 서버 소프트웨어를 설치하십시오.
    2. Linux는, tcputils 패키지로부터 , mini-inetd 를 사용할 수 있습니다.
  2. 프로그램 메뉴에서, 체스 엔진 관리 를 선택하고, 새 네트워크 엔진을 생성 후, 원격 엔진의 호스트 네임과 포트 번호를 입력하세요.
  3. 프로그램 메뉴에서, 체스 엔진 관리 를 선택하고, 체스 엔진 선택 을 누르고, 사용하고자 하는 엔진을 선택하세요.

Gaviota 엔드게임 테이블베이스

Gaviota 엔드게임 테이블베이스 를 사용하려면 :

  1. .gtb.cp4 파일을 SD 카드 상의 DroidFish/gtb 디렉토리에 복사하세요.
  2. 테이블베이스가 어떻게 사용될 지를 조정하려면 엔드게임 테이블베이스 설정을 사용하세요.

Syzygy 엔드게임 테이블베이스

Syzygy 엔드게임 테이블베이스 를 사용하려면 :

  1. .rtbw 그리고 선택적으로 .rtbz 파일을 SD 카드 상의 DroidFish/rtb 디렉토리에 복사하세요.
  2. 테이블베이스가 어떻게 사용될 지를 조정하려면 엔드게임 테이블베이스 설정을 사용하세요.
